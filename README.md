# Køresimulator Prototype

Dette er en prototype af en køresimulator, som er udviklet i Godot.

Formålet med prototypen er at træne trafikregler gennem simple interaktive scenarier. Brugeren klarer baner, hvor forskellige regler introduceres, såsom fartgrænser, stopskilte og trafiklys.

## Brug
Åbn projektet i Godot og kør det fra main scenen.

En spilbar version kan også findes her:  
https://tlcool.itch.io/dansk-kresimulator

## Assets
Dette projekt bruger eksterne assets og udvidelser.
